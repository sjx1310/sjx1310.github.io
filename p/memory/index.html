Memory

1

Time moves in one direction, memory in another.

这是《犯罪心理》一集中的一句话。

Reid的母亲得了阿兹海默症，变得健忘，有时甚至忘记了儿子。Reid则努力地拯救母亲的记忆。

其实剧中关于记忆的描述可以说数不胜数。更典型的是剧中的罪犯。很多人都饱受着记忆不断闪回的折磨，每当过去的片段浮现在眼前，他们痛苦地、不受控制地又一次让自己的双手沾染了鲜血。

2

剧中涉及记忆的不只是罪犯，同样也有受害者或其他当事人。

探员们为了获取关键线索往往会进行一项名为“认知审讯”的活动。

“闭上你的双眼，开始回想……”

然后他们就可以真的回想起一切，可以描述得仿佛就近在眼前。

我一直对这神奇的效果深表怀疑：真的只要闭上双眼，往事就能浮现在眼前吗？尝试过几次，结果是否定的。

后来我才意识到，并不是所有过去的事情都闭上眼就能回忆起来——只有那些重要到能够改变你人生的事情，才足以在你脑海中占据这么重的份量。

或许，那些挥之不去的记忆并不需要回想起来，而是就存放在了那里。闭上眼睛就只是一个必要的程序——就像你打开一个文件夹一样。

3

明白了这件事后一回想，我其实也有过类似的体验。

只不过，闭上双眼的时间是在晚上。那之后我进入的与其说是回忆，用更普遍一点的说法来说，是梦境。

自从上大学以来，我反反复复梦见高考。

4

梦。

想起了质量效应3。那个主角做了3次的梦。背景是逝去朋友们的低语……

5

如果说回忆是存放在大脑之中的文件，那么梦境呢？梦境是不是也可以看作大脑的这样一部分？

至于梦境的不确定性，是否因为它还是一处未经完全探索的领域，还充满着很多未知呢？

Your memory is a monster. It summons at its will. You think you have a memory, but it has you.

6

回忆与现实交织在眼前。

《喧哗与骚动》。

7

忍冬的香气。

勾起回忆的有很多东西，比如香气。

8

我永远不会忘记，课间，教室，她身上的香气。
