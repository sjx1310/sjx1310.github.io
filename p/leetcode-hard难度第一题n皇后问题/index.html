LeetCode Hard难度第一题——N皇后问题

Oct 17, 2020

开始刷LeetCode一个多月了，今天完成了第一道难度为Hard的题目，也是总共第60题。特此纪念一下。

题目描述

n 皇后问题研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

上图为 8 皇后问题的一种解法。

给定一个整数 n，返回 n 皇后不同的解决方案的数量。

示例：

输入: 4 输出: 2 解释: 4 皇后问题存在如下两个不同的解法。 [ [".Q..",
“…Q”, “Q…”, “..Q."],

[”..Q.",
“Q…”, “…Q”, “.Q.."] ]

来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/n-queens-ii。

思路

N皇后是一个很经典的问题了。解决这个问题的方法就是回溯法。

解决一个回溯问题，实际上就是一个决策树的遍历过程。只需要思考 3 个问题：

  1. 路径：也就是已经做出的选择。
  2. 选择列表：也就是你当前可以做的选择。
  3. 结束条件：也就是到达决策树底层，无法再做选择的条件。
result = []
func backtrack(路径, 选择列表):
    if 满足结束条件:
        result.add(路径)
        return

    for 选择 in 选择列表:
        做选择
        backtrack(路径, 选择列表)
        撤销选择

具体到这个问题中：每次摆放一个皇后就是在做出选择。而每个皇后之间都需要满足彼此不能攻击到的条件，这样就限制了摆放好一个皇后之后，下一个的位置。

因此，需要写一个方法来判断两个皇后的位置是否冲突。皇后的攻击范围为上下左右以及对角线。而两个皇后必定不会出现在同一行，所以左右的条件可以直接排除。

接下来判断是否在上下或者对角线，很容易想到利用坐标表示皇后的位置，只需要比较坐标就可以判断是否冲突。

棋盘可以用二维数组 int[][] board来表示吗，这样board[i][j]就可以表示皇后的坐标。

但通过进一步观察可以发现，其实可以把board压缩成一维数组int[] board，这样用数组自身的下标加上在下标处的值也可以表示位置：令board[i]=j，即可表示皇后的坐标为(i,j)。

代码

这样，判断某个位置是否有效的方法如下：

oolean isValid(int[] board, int index){

        for (int i = 0; i < index; i++) {
            //此时正准备放第index个皇后，遍历之前所有已经放下的，判断是否冲突
            if(board[i] == board[index]){
                //表示该位置有皇后位于同一列
                return false;
            }
            if(Math.abs(board[i] - board[index])==Math.abs(index - i)){
                //表示该位置有皇后位于对角线上
                return  false;
            }
        }
        return true;
    }

有了这个方法，就可以写回溯的主方法了。

void backtrack(int[] board, int index, int n){
        if(index == n){
            //如果当前已经循环到第n个，说明棋盘已经摆满了，这样就多了一种有效的解。
            res++;
            return ;
        }
        for(int i = 0 ; i < n ; i++){
            board[index]=i;
            if(isValid(board, index)){
                backtrack(board, index+1,n);
                //递归调用，表示当前位置确实有效，这样就可以放下一排的皇后
            }
        }
    	//如果当前排没有位置有效，说明之前的位置存在错误，这里方法会返回到上一层，也就是摆放上一排的时候。
    }

汇总如下：

class Solution {
    int res = 0;
    public int totalNQueens(int n) {
        int[] board = new int[n];
        backtrack(board,0,n);
        return res;
    }
    void backtrack(int[] board, int index, int n){
        if(index == n){
            res++;
            return ;
        }
        for(int i = 0 ; i < n ; i++){
            board[index]=i;
            if(isValid(board, index)){
                backtrack(board, index+1,n);
            }
        }
    }
    boolean isValid(int[] board, int index){

        for (int i = 0; i < index; i++) {
            if(board[i] == board[index]||Math.abs(board[i] - board[index])==Math.abs(index - i)){
                return  false;
            }
        }
        return true;
    }
}
